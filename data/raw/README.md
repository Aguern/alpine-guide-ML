# Data Raw Directory

This directory is intended for **raw data files** collected from external APIs.

## Why is this directory empty?

Raw data files are **excluded from git** because:
- They are large (100+ MB total)
- They can be regenerated by running data collection scripts
- GitHub has a 100 MB file size limit

## How to populate this directory

Run the data collection scripts to download raw data:

```bash
# Collect all data sources
make collect-data

# Or run individual collectors
python data/ingestion/datatourisme_collector.py
python data/ingestion/insee_melodi_collector.py
python data/ingestion/opendatasoft_collector.py
python data/ingestion/openmeteo_collector.py
```

## Expected files after collection

After running data collection, this directory will contain:

- `datatourisme_pois.parquet` (~40 MB) - Tourism POIs
- `datatourisme_pois_50k.parquet` (~43 MB) - Sample of 50k POIs
- `insee_melodi_salaires.parquet` (~5 MB) - Salary data
- `communes_population.parquet` (~2 MB) - Population data
- `climate_regions.parquet` (~1 MB) - Climate zones
- And other raw data files...

## Note

The processed features ready for ML training are available in:
`data/processed/features_ml.parquet` (included in the repository)
